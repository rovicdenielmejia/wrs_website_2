📝 Step 1: Push Code to GitHub

1 Open VS Code Terminal

Press Ctrl + `

2 Navigate to project

Terminal
git init git add . git commit -m "Initial deployment"
4 Create GitHub Repo
🌐
github.com/new
Name: wrs-website → Click Create repository

Terminal

🗄️ Step 2: Set Up Supabase Database

1 Create Supabase Account
🌐
supabase.com → Sign up → Start your project
2 New Project
➕
Click New project
Name: wrs-database
Password: Create a strong password
Region: Select nearest
⏳ Wait 2-3 minutes for initialization
3 Get Connection String
🔗
Settings → Database → Connection string → Click URI → Copy
Format: postgresql://postgres:PASSWORD@db.xxx.supabase.co:5432/postgres
⚠️ Replace PASSWORD with your Supabase project password!
✅ Done! Database ready with connection string.

🔧 Troubleshooting

Build Failed

• Check all environment variables are set in Vercel
• Verify DATABASE_URL is correct
• Check build logs in Vercel Dashboard

Database Error

• Supabase project not paused
• Password in connection string is correct
• Check Supabase Dashboard

Changes Not Showing

• Push to GitHub: git add . && git commit -m "Update" && git push
• Wait 2-5 min for redeploy
• Clear cache: Ctrl + Shift + R
